## Building
Set `FOUNDRY_VTT_DATA_PATH` if you're using Linux.\
`npm i`\
`npx gulp`

## Installing
Paste "https://raw.githubusercontent.com/gulbanana/fmmua/master/src/system.json" into Foundry's Install System window.
